Lou-extensions

Extension manager for Lord Of Ultima

View project on GitHub

This project aims to provide basic functionality to the community for writing their own Chrome extensions and GreaseMonkey scripts for Lord Of Ultima (LoU).

Features

  • Dynamic script loader to inject and eject scripts on the fly.
  • A modified LoU BoS 1.6.0 using these features.
  • Backwards compatibility layer to ensure older LoU scripts keep functioning. Confirmed:
    • LoU Suite (76 Edit)
    • The Defiant (TDK) Extension

Install latest lou-extensions

This version includes a set of functions that recreates functions that were removed in the recent patch that upgraded the game from Qooxdoo 1.4 to Qooxdoo 2.0.1. The compatibility functions make sure that older versions of LoU BoS, LoU Tweak, LoU Suite and others continue working without requiring an update from their original authors. View the change log on the project wiki.

Download lou-extension version 0.6.3 for Chrome

Installation steps:

  1. Download and save the chrome extension to your local disk and open the download folder.
  2. In Chrome go to the extensions page: menu --> wrench icon --> tools --> extensions
  3. This opens the Chrome extensions page chrome://chrome/extensions/
  4. Drag the downloaded extensions from your local disk onto the extensions page

Older versions

Extension installed from this section do not use auto-update features. Install from here if you don't want to follow the automatic update cycle. This could be useful in case a newer version doesn't work

install lou-extension version 0.5.0 for Chrome (release date 24th July 2012)
